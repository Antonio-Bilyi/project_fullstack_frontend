# Подорожники

## Опис

**Подорожники** — це вебсайт, де користувачі можуть ділитися власними історіями про подорожі та відкривати нові цікаві місця.  
Мета проєкту — створити платформу для обміну досвідом між мандрівниками та надихати інших на нові пригоди.

## Використані технології

- **Frontend:** Next.js, React, TanStack Query, Axios, Formik, Yup, Zustand, TypeScript
- **Backend:** Node.js, Express, Multer, Cloudinary, Joi, bcrypt, dotenv, Swagger
- **Database:** MongoDB
- **Інше:** Cookie, React Hot Toast

## Запуск проєкту

1. Клонуйте репозиторій:
   ```bash
   git clone git@github.com:Antonio-Bilyi/project_fullstack_frontend.git
   ```
2. Перейдіть у папку проєкту:
   ```bash
   cd podorozhnyky
   ```
3. Встановить залежності:
   ```bash
   npm install
   ```
4. Запустіть локальний сервер:
   ```bash
   npm run dev
   ```
5. Відкрийте у браузері: http://localhost:3000
   За потреби може знадобитися налаштування змінних середовища (наприклад, для підключення до MongoDB).
